Fügt Labels zu Jenkins-Agenten hinzu, die auf den Eigenschaften des Betriebssystems mit dem der Agent ausgeführt wird bestimmt werden.

Die Labels enthalten standardmäßig Betriebssystem, Version und Architektur. Die Erstellung von Etiketten kann global über die Seite "System Configuration" von "Jenkins verwalten" gesteuert werden. Die Erstellung von Labels kann auf jedem Agenten von der Seite "Konfigurieren" jedes Agenten kontrolliert werden. Label-Einstellungen des Agenten haben Vorrang vor den globalen Agenten-Gabeleinstellungen.
